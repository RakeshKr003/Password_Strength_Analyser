# 🔐 Password Strength Analyzer

A simple Python script to analyze the strength of a password based on common security rules.  
This tool provides quick feedback to help users create stronger, more secure passwords.

---

## 📋 Features
- Checks if the password:
  - Has at least **8 characters**
  - Contains at least **one uppercase letter**
  - Contains at least **one lowercase letter**
  - Contains at least **one digit**
  - Contains at least **one special character** (`@ # $ % ^ & * !`)
- Returns clear, user-friendly messages:
  - Explains why a password is weak
  - Confirms when a password is strong

---

## 🛠️ Requirements
- Python 3.13.12 or above
- Standard library (`re` module is used, no external dependencies)
